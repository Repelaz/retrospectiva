CSS

Essa sem dúvidas foi a melhor matéria na minha opinião, mas o que foi aparendido em CSS? Diversas coisas sobre a personalização do site, sendo os códigos: Border-radius, padding que serve para alterar dentro do objeto, margin que serve para alterar fora do objeto, border para colocar borda no objeto entre outros como background para alterar o fundo de um objeto, color para alterar a cor da escrita e o melhor o br que serve para pular uma linha.

HTML

O HTML é uma linguagem de marcação que define a estrura do conteúdo na web. Nessa parte aprendemos a estrura base do HTML, que é a seguinte, no HTML principal será colocado a direção que o site será lido e a linguagem do site, a outra parte é o head, que na tradução literal seria a cabeça do site, é aqui onde estão todas as funções não visíveis do site, como por exemplo o programa que linka com o CSS, a ultima parte é o body, o corpo do site, onde ficam todas as coisas visíveis do site, é aqui onde é estruturado o site, título, header e footer.
